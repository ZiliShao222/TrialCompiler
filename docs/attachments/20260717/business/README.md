## 使用边界

原附件中的行业判断、企业内部参数和 POC 目标值不能自动视为已验证事实。比赛材料引用前，应以结构化核验表中的 `evidence_status` 为准。

新提交的用户反馈与专家访谈材料同样属于内部汇编。社交语料在补齐原始 URL、截图或存档前不得作为定量用户证据；专家条目应以核验报告中的修正摘要为准。
