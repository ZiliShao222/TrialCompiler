# 完全合成变更请求：PK 最终采样时间 32 h → 36 h

**测试资产编号：** TC-MUT-001  
**状态：** 完全合成，仅用于 TrialCompiler 原型验证  
**不得解释为 NCT04683926 的真实修订。**

## 变更内容

将每个处理期的最终 PK 血样名义采样时间由给药后 **32 h** 调整为 **36 h**。其他采样时间保持不变。

原序列：0、0.5、1、1.5、2、2.5、3、3.5、4、6、8、12、16、24、32 h。  
候选新序列：0、0.5、1、1.5、2、2.5、3、3.5、4、6、8、12、16、24、36 h。

## 预期影响对象

Protocol Synopsis、Study Design/Procedures、Appendix A Schedule of Procedures、SAP 中 PK 数据窗口与 TFL 规范、ICF 中采血与住院负担说明、合成 CRF 的名义时间点允许值、ClinicalTrials.gov 结果 time frame，以及任何 AUC0-t/AUC0-inf 计算和缺失样本判定规则。

## 预期系统行为

系统先创建候选事实新版本，保留原 32 h 值；根据依赖图生成受影响对象清单；对确定性位置生成候选红线；对 ICF 住院时长、AUC 计算和 PP 样本完整性规则发起统计、临床药理、临床运营和伦理复核；未经批准不得覆盖正式值。

## 验收标准

至少命中依赖边 E001–E005。所有候选修订均带来源、旧值、新值、传播路径、责任角色和重检状态。系统不得静默修改正式文档。
